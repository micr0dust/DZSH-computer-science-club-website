C++

C++是一種被廣泛使用的電腦程式設計語言。它是一種通用程式設計語言，支援多重程式設計模式，例如程序化程式設計、資料抽象化、物件導向程式設計、泛型程式設計和設計模式等。 比雅尼·斯特勞斯特魯普博士在貝爾實驗室工作期間在20世紀80年代發明並實現了C++。


JavaScript

JavaScript是一種進階的、直譯的程式語言。JavaScript是一門基於原型、函式先行的語言，是一門多範式的語言，它支援物件導向編程，指令式程式設計，以及函式語言程式設計。它提供語法來操控文字、陣列、日期以及正規表示式等，不支援I/O，比如網路、儲存和圖形等，但這些都可以由它的宿主環境提供支援。


Python

Python是一種廣泛使用的直譯式、進階程式、通用型程式語言，由吉多·范羅蘇姆創造，第一版釋出於1991年。可以視之為一種改良的LISP。Python的設計哲學強調代碼的可讀性和簡潔的語法。相比於C++或Java，Python讓開發者能夠用更少的代碼表達想法。不管是小型還是大型程式，該語言都試圖讓程式的結構清晰明了。

社長
當初突然獲得了兩個我不怎麼擅長的職位，雖然後來發現社團內需要我的地方不多但也還是努力試著做了自己該做的工作，後果不太好說但也是一個經驗！ 就我在社團的感覺校外的交流雖然不是沒有但從我也是社員開始就沒有很注重這個，所以身為公關都是在校內活動，不過也有個人比較怕生的原因，不是一個很合適的公關人選。美宣的部分嘗試幫社團繪製了社服的圖案，遇到了各種不盡理想的狀況所以從此以後沒什麼信心。之後就當個助手幫忙社團事務的推進和執行，個人覺得自己雖然在職位上沒有什麼用處但在社團內還是有不少用處的，希望社團之後能克服我們遇到的問題並成長茁壯。
副社長
我是資訊社的副社長蕭傳胤，也是大直數理班的同學，個性比較內向，不會主動和別人聊天。在社團擔任副社長，主要的工作是負責寫寫社團記錄本，紀錄教學內容和各社團成員間的互動，或者有特殊活動時，要幫忙擔任助手或拍攝，另一項工作是有需要的話會幫忙請公假。我還擔任副教學，平時教學長教程式時，我在台下幫忙學弟妹解決一些問題，或者他們聽不懂時幫忙解釋。我負責教學弟妹一些電路板的基本概念，也介紹到Autodesk arduino ，這個網站可以模擬電路，裡面也有arduino 板，可以在這塊板子上寫一些指令。在資訊社這段學到很多平常不會接觸到知識，增加自己的內涵 。

總務
還沒當過股長時，總是以為當股長其實很輕鬆，或是會有些什麼特權；但是當自己當股長時，就會發現當股長也沒有那麼容易。像是班長就要處理班上的大小事務，而總務股長更是一件難當的差事，因為擔任股長若做的不好，還會遭人閒話與責備。只能說，沒有當過股長，真的不會知道股長們的辛苦。而擔任資訊社總務職位的時候，其實是滿害怕的，因為不知道自己是否能夠勝任，總覺得光『總務』這兩個字，好像就要背負很大的壓力跟責任，畢竟要管理與經營一個社團比管理一個班級還要來得不容易。但是幸好有學姐在旁協助指導，讓我覺得輕鬆不少。由於當時的我不太擅長和不熟悉的人接洽，總會不知所措而一團亂，但還好學姐與社員都會互相幫忙，讓我很感謝大家。也著實讓我感覺：社團也是一個小社會、也是一個小家庭。想想過去我們一直互相扶持的比過數場比賽
